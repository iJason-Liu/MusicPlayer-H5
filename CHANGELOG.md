# 📝 更新日志

## [1.0.0] - 2024-10-22

### ✨ 新增

#### 后台管理

- 🎵 音乐管理功能（增删改查）
- 📊 统计面板（音乐总数、匹配率、存储空间、总时长）
- 🔍 自动扫描音乐目录功能
- 🎯 批量匹配音乐信息（封面、歌词、专辑）
- ⬆️ 音乐文件上传功能（支持 MP3、FLAC、WAV、M4A）
- 📦 批量操作（批量匹配、批量删除）
- 💾 音乐信息缓存机制（7天）
- 🔄 最近添加音乐列表

#### 前端播放

- 🎧 音乐播放功能（支持多种格式）
- 🎵 音乐库页面（列表展示、搜索）
- 🎮 全屏播放器页面
  - 旋转封面动画
  - 歌词显示
  - 进度条控制
  - 播放控制按钮
- 💾 播放列表管理
- 📜 播放历史记录
- 💖 收藏功能（我的喜欢）
- 🎭 三种播放模式（列表循环/随机播放/单曲循环）
- 📱 迷你播放器（底部悬浮）
- 🎨 液态玻璃 UI 设计（苹果风格）
- 🌈 渐变背景主题
- 📱 完美适配移动端

#### 技术特性

- ⚡ Vue 3 Composition API
- 🗃️ Pinia 状态管理
- 🎯 Vue Router 路由管理
- 📦 Vant 4 UI 组件库
- 🎨 SCSS 样式预处理
- 🔄 本地存储持久化（播放列表、历史、收藏）
- 🚀 Vite 5 快速构建
- 📡 Axios HTTP 请求
- 🎵 网易云音乐 API 集成

## 版本说明

### 版本号格式

`主版本号.次版本号.修订号`

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 更新类型

- ✨ **新增** - 新功能
- 🐛 **修复** - Bug 修复
- 🎨 **优化** - UI/UX 改进
- ⚡ **性能** - 性能优化
- 🔧 **配置** - 配置变更
- 📚 **文档** - 文档更新
- 🔒 **安全** - 安全修复
- ♻️ **重构** - 代码重构
- 🗑️ **移除** - 功能移除
- 🚨 **破坏性变更** - 不兼容的变更
